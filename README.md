# Nexora

> Agence créative fictive spécialisée en développement web, design graphique, branding et communication digitale.

## À propos

Nexora est un projet réalisé dans le cadre de la **Semaine 5** de la formation **Akieni Academy - Cohorte 2**.

L'objectif est de concevoir un site vitrine moderne, responsive et entièrement développé en **HTML5** et **CSS3**, tout en appliquant les bonnes pratiques de collaboration avec **Git** et **GitHub**.

## Buts

- Présenter l'agence
- Mettre en avant les réalisations
- Présenter l'équipe
- Permettre aux visiteurs de contacter l'agence

## Workflow Git

Chaque membre travaille sur une branche dédiée.

Exemple :

```bash
feature/home
feature/team
feature/portfolio
feature/contact
```

Toutes les modifications passent par une **Pull Request** avant d'être fusionnées dans la branche **main**.

## Responsive

Le site est optimisé pour :

- Mobile (375px)
- Tablette (768px)
- Desktop (1280px)

## Conflit Git

Un conflit de fusion sera volontairement rencontré puis résolu durant le projet.
La procédure de résolution sera documentée dans ce README.

### Résumé

- Conflit rencontré lors du merge de deux Pull Requests.
- Résolution effectuée manuellement.
- Vérification avant le merge final.
